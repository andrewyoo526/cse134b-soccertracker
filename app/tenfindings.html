Our Ten Findings

  1. MathML isn't supported by Chrome, Edge, IE. But is supported by Safari, and FireFox
  2. It's cool how an href attribute can handle both local files and external urls
  3. The importance of 'meta' tags for SEO and the undervalued and unknown this tag is
  4. There are many ways to manipulate the text on the page such that you see words crossed out, emphasized words, inserted words and even texts that are linked
  5. W3schools isn't always reliable, sometimes their examples are not in properly formed html
  6. There was much more inconsistencies between browsers on implementation and support than expected
  7. Most of the tag attributes related with visual design are not longer supported in HTML5 because it was redundant (for those purposes we use CSS)
  8. Bold and Strong tags do not have the same purpose. While in bold you just want to highlight a word, strong is an indication of how something should be understood. It can mean "bold" in a browser, but it is also represented in some way for PDA's or devices for blind people
  9. You can place Javascript from events in the script section within the html, which holds code similar to other programming languages.
  10. Different event functions can occur depending on how the user interacts with the elements of the page. This includes the user clicking on buttons, pressing keys, or even just loading the page.
